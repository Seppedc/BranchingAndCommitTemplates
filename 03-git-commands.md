## Git commands 
Usefull Git commands 

| **Naming Convention**     | **Purpose**                                                              |
| ------------------------- | ------------------------------------------------------------------------ |
| `git reset --soft HEAD~1` | Undo the last git commit                                                 |
| `git push --force`        | Force push local commit onto the branch, wich ovverides incomming commits|
| `git rebase -i HEAD~2`    | Locally check last 2 commits, then use vm editor to remove a commit      |
| `vm editor`               | use ESC, :wq , Enter to save and quit vm editor                          |
